# Frontier Health

A comprehensive healthcare compliance and clinical data management platform built on FHIR R4 standards.

## Features
- Patient management with FHIR R4 Patient resources
- Clinical records: Observations (vitals, labs) and Conditions
- Encounter tracking
- Medication management
- FHIR-compliant data models (US Core aligned)

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- FHIR R4 (@types/fhir)
- Recharts for data visualization

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
